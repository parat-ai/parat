# Project Instructions

## Getting Started

Before working on this project, start a role:

| Role | Command | Description |
|------|---------|-------------|
| Executor | `/executor [n]` | Execute tickets and write code |
| Planner | `/planner [n]` | Plan features and create tickets |
| Bughunter | `/bughunter [n]` | Review code and find issues |
| Writer | `/writer [n]` | Write documentation |

Where `[n]` is an optional instance number (1-9). Example: `/executor 2`

## Project Structure

- Ticket data is stored in the parent `ticket-data/` directory
- Environment config is in `.env`
